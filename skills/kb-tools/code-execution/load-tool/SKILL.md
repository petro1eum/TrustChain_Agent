# Load Tool Tool

Инструмент для загрузки кода инструмента по имени.

## Параметры

- `toolName` (required, string) - Имя инструмента

## Использование

Используй для:
- Просмотра кода инструмента
- Модификации существующих инструментов
- Понимания работы инструмента

## Примеры

```json
{
  "toolName": "analyze-diagnostics"
}
```

## Результат

Возвращает код инструмента как строку.
