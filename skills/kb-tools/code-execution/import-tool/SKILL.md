# Import Tool Tool

Инструмент для импорта и выполнения инструментов из файлов.

## Параметры

- `toolPath` (required, string) - Путь к файлу инструмента (например, '/agent-tools/my-tool.ts')

## Использование

Используй для:
- Использования сохранённых инструментов
- Переиспользования кода
- Выполнения специализированных инструментов

## Примеры

```json
{
  "toolPath": "/agent-tools/analyze-diagnostics.ts"
}
```

## Результат

Возвращает результат выполнения импортированного инструмента.
