# Execute Bash Tool

Инструмент для выполнения bash команд в безопасной изолированной среде.

## Параметры

- `command` (required, string) - Bash команда для выполнения
- `working_dir` (optional, string) - Рабочая директория (относительно agent_workspace)
- `timeout` (optional, number) - Таймаут выполнения в секундах (1-300, по умолчанию 30)

## Безопасность

- Команды выполняются в изолированной среде
- Опасные команды запрещены (rm, mkfs, sudo и т.д.)
- Whitelist разрешённых команд
- Ограничение размера вывода
- Timeout для предотвращения зависаний

## Сохранение состояния

Рабочая директория сохраняется между командами, что позволяет:
- Создавать файлы в одной команде
- Использовать их в следующих командах
- Сохранять состояние между вызовами

## Примеры

```json
{
  "command": "ls -la"
}
```

```json
{
  "command": "pwd"
}
```

```json
{
  "command": "cat file.txt",
  "working_dir": "data"
}
```

## Результат

Возвращает результат выполнения команды с stdout, stderr и returncode.
