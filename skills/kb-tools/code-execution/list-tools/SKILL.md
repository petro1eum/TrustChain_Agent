# List Tools Tool

Инструмент для получения списка всех доступных инструментов.

## Параметры

Нет параметров.

## Использование

Используй для:
- Просмотра доступных инструментов
- Поиска нужного инструмента
- Понимания доступных возможностей

## Пример использования

```json
{}
```

## Результат

Возвращает список сохранённых инструментов с:
- Имя инструмента
- Путь к файлу
- Описание (если доступно)
