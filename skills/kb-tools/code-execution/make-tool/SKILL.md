# Save Tool Tool

Инструмент для сохранения созданных инструментов как TypeScript файлов.

## Параметры

- `toolName` (required, string) - Имя инструмента (без расширения)
- `code` (required, string) - Код инструмента (TypeScript/JavaScript)
- `basePath` (optional, string) - Базовый путь для сохранения (по умолчанию '/agent-tools')

## Использование

Используй для:
- Сохранения полезных инструментов
- Создания библиотеки переиспользуемых инструментов
- Организации кода

## Примеры

```json
{
  "toolName": "analyze-diagnostics",
  "code": "export function analyzeDiagnostics(results) { ... }"
}
```

## Результат

Возвращает статус сохранения инструмента и путь к файлу.
