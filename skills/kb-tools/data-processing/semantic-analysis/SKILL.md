# Semantic Analysis Tool

Инструмент для семантического анализа текстовых данных с использованием GPT-4.

## Параметры

- `column` (required, string) - Колонка с текстом для анализа
- `instruction` (required, string) - Инструкция для анализа (на русском языке)
- `new_column` (required, string) - Название новой колонки для результата

## Использование

Инструмент использует GPT-4 для анализа текста согласно инструкции. Инструкция должна быть чёткой и конкретной.

## Примеры инструкций

- "Извлеки название компании из текста"
- "Определи тональность отзыва (положительная/отрицательная/нейтральная)"
- "Классифицируй запрос по категориям: техническая поддержка, продажи, жалоба"
- "Извлеки ключевые слова из текста"

## Пример использования

```json
{
  "column": "feedback",
  "instruction": "Определи тональность отзыва: положительная, отрицательная или нейтральная",
  "new_column": "sentiment"
}
```

## Результат

Возвращает:
- `success` - статус выполнения
- `new_column` - имя новой колонки
- `analysis` - результат анализа от GPT-4
