# Text Processing Tool

Инструмент для обработки и очистки текстовых данных в workspace.

## Параметры

- `column` (required, string) - Название колонки для обработки
- `operations` (required, array) - Список операций для применения

## Доступные операции

- `clean` - Очистка текста: удаление лишних пробелов, специальных символов (оставляет только буквы, цифры, дефисы, точки, @)
- `lowercase` - Преобразование в нижний регистр
- `uppercase` - Преобразование в верхний регистр
- `strip` - Удаление пробелов в начале и конце строки
- `extract_emails` - Извлечение email адресов из текста
- `extract_phones` - Извлечение телефонных номеров из текста

## Использование

Операции применяются последовательно в порядке указания. Результат сохраняется в новую колонку с суффиксом `_processed`.

## Примеры

```json
{
  "column": "name",
  "operations": ["strip", "lowercase"]
}
```

```json
{
  "column": "contact_info",
  "operations": ["extract_emails", "extract_phones"]
}
```

## Результат

Возвращает:
- `success` - статус выполнения
- `column` - обработанная колонка
- `operations` - применённые операции
- `processed_count` - количество обработанных строк
- `new_column` - имя новой колонки с результатами
- `sample_before` - примеры данных до обработки
- `sample_after` - примеры данных после обработки
