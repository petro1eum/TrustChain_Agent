# Analyze Data Quality Tool

Инструмент для комплексного анализа качества данных в workspace.

## Параметры

- `full_analysis` (optional, boolean) - Полный анализ или быстрый обзор (по умолчанию true)

## Что анализируется

- Общая статистика (количество строк, колонок)
- Пропущенные значения (общее количество и по колонкам)
- Дубликаты (полные и частичные)
- Типы данных
- Выбросы в числовых колонках
- Рекомендации по улучшению

## Примеры

```json
{
  "full_analysis": true
}
```

```json
{
  "full_analysis": false
}
```

## Результат

Возвращает объект с анализом:
- `total_rows` - общее количество строк
- `total_columns` - общее количество колонок
- `missing_values` - информация о пропусках:
  - `total` - общее количество пропусков
  - `by_column` - пропуски по колонкам
- `duplicates` - информация о дубликатах:
  - `full_duplicates` - полные дубликаты
  - `partial_duplicates` - частичные дубликаты
- `data_types` - типы данных по колонкам
- `outliers` - информация о выбросах
- `recommendations` - рекомендации по улучшению качества данных
