# Handle Missing Data Tool

Инструмент для обработки пропущенных значений в данных.

## Параметры

- `strategy` (required, string) - Стратегия обработки пропусков
- `group_by` (optional, array) - Колонки для группировки при обработке
- `n_neighbors` (optional, integer) - Количество соседей для KNN (по умолчанию 5)

## Доступные стратегии

- `drop` - Удалить строки с пропусками
- `fill_mean` - Заполнить средним значением
- `fill_median` - Заполнить медианой
- `fill_mode` - Заполнить модой (наиболее частое значение)
- `interpolate` - Интерполяция значений
- `knn` - KNN imputation (требует `n_neighbors`)

## Примеры

```json
{
  "strategy": "fill_mean"
}
```

```json
{
  "strategy": "knn",
  "n_neighbors": 10,
  "group_by": ["category"]
}
```

## Результат

Возвращает:
- `success` - статус выполнения
- `strategy` - применённая стратегия
- `result` - детали выполнения
- `groups` - группы для групповой обработки
