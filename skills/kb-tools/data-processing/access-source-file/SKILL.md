# Access Source File Tool

Инструмент для доступа к исходным файлам с фильтрацией данных.

## Параметры

- `filename` (required, string) - Имя файла для доступа
- `columns` (optional, array) - Список колонок для выборки
- `condition` (optional, string) - Условие фильтрации данных

## Использование

Инструмент позволяет читать данные из ранее загруженных исходных файлов с возможностью выборки нужных колонок и применения фильтров.

## Примеры

```json
{
  "filename": "products.csv"
}
```

```json
{
  "filename": "orders.csv",
  "columns": ["order_id", "date", "total"],
  "condition": "total > 1000"
}
```

## Важно

- Файл должен быть предварительно загружен в контекст
- Условие фильтрации должно быть валидным Python выражением
- Если указаны `columns`, возвращаются только выбранные колонки

## Результат

Возвращает:
- `success` - статус выполнения
- `data` - отфильтрованные данные
- `rowCount` - количество строк в результате
