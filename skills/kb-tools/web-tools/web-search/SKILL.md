# Web Search Tool

Инструмент для выполнения веб-поиска через DuckDuckGo.

## Параметры

- `query` (required, string) - Поисковый запрос
- `maxResults` (optional, number) - Макс. число ссылок (1-10, по умолчанию 10)

## Использование

Используй для:
- Поиска актуальной информации
- Поиска документации
- Поиска примеров и решений

## Примеры

```json
{
  "query": "OpenSearch query syntax",
  "maxResults": 5
}
```

## Результат

Возвращает список URL с результатами поиска.
