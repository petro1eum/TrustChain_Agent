# Web Fetch Tool

Инструмент для загрузки содержимого веб-страницы.

## Параметры

- `url` (required, string) - URL страницы

## Использование

Используй после `web_search` для:
- Получения содержимого найденных страниц
- Чтения документации
- Извлечения информации с веб-страниц

## Примеры

```json
{
  "url": "https://opensearch.org/docs/latest/query-dsl/"
}
```

## Результат

Возвращает текстовое содержимое страницы.
