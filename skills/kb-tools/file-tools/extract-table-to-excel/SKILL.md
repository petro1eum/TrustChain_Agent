# Extract Table To Excel

Инструмент для извлечения табличных данных из PDF-вложения по заданным страницам с сохранением в Excel.

## Параметры

- `filename` (required, string) — имя PDF файла из блока [ВЛОЖЕНИЯ]
- `page_start` (required, number) — начальная страница (1-based)
- `page_end` (required, number) — конечная страница (1-based)
- `sheet_name` (optional, string) — имя листа Excel
- `user_query` (optional, string) — запрос пользователя (без искажений)

## Когда использовать

- Пользователь просит выгрузить таблицу/товары из PDF в Excel
- Указан диапазон страниц (например, 17–20)

## Пример

```json
{
  "filename": "spec-20240101-abc.pdf",
  "page_start": 17,
  "page_end": 20,
  "sheet_name": "Товары"
}
```

## Результат

Возвращает ссылку на Excel файл и вложение для чата.

## Формат данных

Рекомендуемый JSON от модели:

```json
{ "rows": [ { "колонка1": "...", "колонка2": "..." } ] }
```
